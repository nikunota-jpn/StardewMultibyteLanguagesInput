# StardewMultibyteLanguagesInput
This is a utility for entering multibyte characters in the PC game ‘Stardew Valley’.

* * *
Nexus Mods Page ** LINK **
* * *

## Overview
This is an input assist tool for StardewValley. It supports the input of multibyte text required by the in-game IME.

## Requirements
To run this tool, you must have the following installed:

  .NET 8.0 Desktop Runtime (x64 or x86 depending on your OS)

## Installation & Usage
1. Download the latest version from Nexus Mods or the GitHub Releases page.
2. Extract the ZIP file to any folder.
3. Run StardewMultibyteLanguagesInput.exe.
4. Launch Stardew Valley, open a screen where text input is required, enter the text into this tool, and press Enter. The text will be automatically copied into the game.

## 概要
StardewValley用の入力補助ツールです。ゲーム中のIMEが必要なマルチバイトテキスト入力をサポートします。

## 動作環境
本ツールの実行には、以下のランタイムが必要です。
  .NET 8.0 デスクトップ ランタイム

## 使い方
1. Nexus Mods または GitHubのリリース詳細 から最新版をダウンロードしてください。
2. ZIPファイルを任意の場所に展開します。
3. StardewMultibyteLanguagesInput.exe を実行してください。
4. StardewValleyを起動し、テキスト入力が求められる画面を開き、このツールにテキストを入力してEnterキーを押してください。テキストが自動的にゲーム内にコピーされます。
